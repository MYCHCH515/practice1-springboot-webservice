## SpringBoot 예제 학습
참고: https://github.com/jojoldu/freelec-springboot2-webservice

책: 스프링 부트와 AWS로 혼자 구현하는 웹 서비스

+ 예제 내용 :
  + 게사글 등록/조회/삭제/수정
  + 소셜로그인(구글 / 네이버 로그인)
  + AWS EC2/ AWS RDS 환경 설정
  + 배포 자동화 
  + 무중단 배포 
+ 개발 환경 : IntelliJ, SpringBoot, Jdk 8, Junit4, Gradle 4.8
+ 웹 서버 : AWS EC2
+ 데이터 베이스 : AWS RDS
+ 협업 도구 : Git
